Configure how local AI clients (like Claude Desktop) connect to ProjectBrain. You can choose to run MCP via local Stdio (spawning a process) or HTTP/SSE (server-sent events stream).

Integration Settings

Stdio mode registers a command calling ProjectBrain's package on this python environment directly, setting these default tags and user ID in environment variables.

SSE mode routes client queries to the running server. It registers a proxy connection using mcp-remote inside Claude Desktop.
